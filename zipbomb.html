                Map tools
                

                    Edit the map text below and click Load. Symbols: # wall, @ player, v/^/</> enemies,
                    $ coin, Z victory (appears only after all coins are collected), and . or space for floor.

                    Optional header lines at the very top start with ; for settings, e.g.
                    ; enemyTick=200 (also accepts ; tick=200) to set enemy speed in ms.
